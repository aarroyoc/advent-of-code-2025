# advent-of-code-2025

My solutions for AoC 2025 in Scala

Needs Scala CLI (Scala installed through Coursier has it)

Run everything

```
scala .
```

Run tests

```
scala test .
```

Execute REPL

```
scala repl .
```

Generate JAR

```
scala --power package . --assembly --preamble=false
```

Native compilation

```
scala --power package . --graal (GraalVM)
scala --power package . --native (Scala Native)
```
